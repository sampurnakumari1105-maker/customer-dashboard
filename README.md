Project Title

Customer Demographics & Sales Analysis Dashboard

📝 Short Description

This project is an interactive dashboard designed to analyze customer demographics and sales performance. It provides insights into total sales, customer count, and sales distribution across different demographics such as gender, marital status, age groups, and geographic locations.

🛠️ Tech Stack
Data Visualization Tool: Power BI / Tableau (most likely Power BI based on layout)
Data Processing: Excel / SQL / Python (for data cleaning and transformation)
Database (optional): SQL Server / MySQL
Map Integration: Bing Maps (used in the dashboard for geographic visualization)
Data Source
Retail or e-commerce dataset (sample or real)
Likely includes fields such as:
Customer ID (CustomerKey)
Sales Amount
Gender
Marital Status
Age / Age Band
Country / Region
Product Name
Key Features
KPI Cards:
Total Sales (29.36M)
Total Customers (18.48K)
Demographic Analysis:
Sales distribution by Gender
Sales by Marital Status
Sales by Age Band (bar chart)
Geographical Insights:
World map showing Total Sales by Country
Interactive Filters:
Product Name filter (top right)
Visualizations Used:
Donut charts
Bar charts
KPI cards
Map visualization
Purpose of the Project
To understand customer behavior and buying patterns
To identify high-performing regions and demographics
To support business decision-making using data-driven insights
